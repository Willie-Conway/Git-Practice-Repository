2. **Create a Branch (optional)**:
   
```bash

```
3. **Make Changes and Commit**:

- Edit files as needed.
- Stage your changes:
  
```bash
git add .
```
4. **Commit your changes**:
   
```bash
git commit -m "Your commit message here"
```
5. **Push to GitHub**:

```bash
git push origin practice-branch
```
